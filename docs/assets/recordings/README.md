# ChoreStar Recording Guide

Place screen recordings (GIFs or short videos) here. Use Kap, Cmd+Shift+5, or Loom.

---

## 1. Create a Routine

**Path:** Dashboard → (child selected) → Add Routine or Routines section

**Steps to record:**
1. Open the dashboard with a child selected
2. Click **Add Routine** (or navigate to the routines section)
3. Enter routine name (e.g., "Morning Routine")
4. Choose type (Morning / Bedtime / Afterschool / Custom)
5. Add steps (title, optional description, icon)
6. Set reward amount
7. Save the routine

**Suggested filename:** `create-routine.gif` or `create-routine.mov`

---

## 2. Kid Login

**Path:** `/kid-login/[familyCode]` (share link from Settings → Family)

**Steps to record:**
1. Start on the kid login page (use the family link from Settings → Family)
2. Show the numeric keypad
3. Enter the child's 4–6 digit PIN
4. Show the transition to the kid dashboard (child's name, routines)

**Suggested filename:** `kid-login.gif` or `kid-login.mov`

---

## 3. Run a Routine

**Path:** Kid dashboard → Select routine → Complete steps

**Steps to record:**
1. Start on the kid dashboard (after PIN login)
2. Show the list of routines (Morning, Bedtime, etc.)
3. Tap a routine to open it
4. Step through each checkbox (tap to complete steps)
5. Show the completion / celebration screen

**Suggested filename:** `run-routine.gif` or `run-routine.mov`
