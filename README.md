# ChoreStar

A modern family chore tracking app that turns household tasks into a game kids love. Built with Next.js 15, Supabase, and Stripe.

**Live:** [chorestar.app](https://chorestar.app)

## Project Structure

```
family-chore-chart/
├── chorestar-nextjs/    ← Active application (Next.js 15)
├── frontend/            ← Legacy vanilla JS app (archived)
├── backend/             ← Supabase schema & functions
└── docs/                ← Assets & recordings
```

See [`chorestar-nextjs/README.md`](./chorestar-nextjs/README.md) for setup instructions and full documentation.
